ARMA Representation (Francq, Zakoian)¶

canonical SV Model¶

$ \epsilon_t = \sqrt{h_t} \eta_t \\ log(h_t) = \omega + \beta log(h_{t-1}) + \sigma v_t $

$ \eta_t \sim \mathcal{N}(0,1) \\ v_t \sim \mathcal{N}(0,1) $

State-space model¶

$ log (\epsilon_t^2) = log (h_t) + \mu_Z + u_t \\ log (h_t) = \beta log (h_{t-1}) + \omega + \sigma v_t $
